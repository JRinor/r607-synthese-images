IN6 - Canevas du TP sur la dicrétisation de segments droits

1ere partie : Tracé équilibré

Compiler et exécuter le fichier Trace.java.
Choisir deux pixels dans la grille à l'aide de la souris. Un troisième clic est sensé déclencher le tracé du segment joignant les deux points. Ce tracé est pour l'instant inopérant.
Implanter l'algorithme de tracé arithmétique présenté en cours dans la méthode tracerSegment du fichier Pixel.java.

Le code pourra être testé avec la classe Test.
En bleu, les pixels justes.
En jaune, les pixels oubliés.
En vert, les pixels faux.
En rouge, les pixels tracé plusieurs fois.
Taper 1, 2, 3 ou 4 pour quelques cas standard (non exhaustifs).
2eme partie : Tracés basés région
Implanter les algorithmes de tracé correspondant dans la méthode tracerSegment avec un paramètre d'équilibrage dans Pixel.java.
La touche 'r' force les tracés en discrétisation basée région supérieure. Le paramètre eq de tracerSegment a la valeur +1.
La touche 'z' force les tracés en discrétisation basée région inférieure. Le paramètre eq de tracerSegment a la valeur -1.
La touche 'e' force les tracés en discrétisation équilibrée (standard). Le paramètre eq de tracerSegment a la valeur 0.
